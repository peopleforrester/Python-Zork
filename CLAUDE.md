# Python Zork

Educational text-based adventure teaching computer architecture

**Stack**: Python, Node.js/TypeScript, React

## Commands

- **Install**: `uv sync --dev`
- **Test**: `uv run pytest`
- **Lint**: `uv run ruff check .`
- **Format**: `uv run ruff format .`
- **Type check**: `uv run mypy src/`
- **Install**: `npm install`
- **Dev**: `npm run dev`
- **Build**: `npm run build`
- **Test**: `npm test`
